# v1.0.0

First public release of **Codex Usage** for Ulanzi Deck.

- 5-hour and Weekly limit actions (Claude-style gauges)
- Reads ChatGPT Codex login from `~/.codex/auth.json`
- Fetches usage from ChatGPT `wham/usage`
- Auto-refreshes OAuth access tokens on 401/403
- Codex product mark icons on buttons and in the action palette
